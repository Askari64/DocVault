## Frontend Setup Instructions

### Prerequisites
- Node.js (v18+)
- Clerk account (for authentication)

### Installation

```bash
# Clone repository
git clone <your-repo-url>
cd frontend

# Install dependencies
npm install

# Configure environment
cp .env.example .env
# Edit .env with your Clerk keys

# Start development server
npm run dev
